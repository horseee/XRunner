## XRunner

An android app with AR. 

### Install

* SDK: 23
* In AndroidManifest.xml, please modify the following line. The key can be applied on ***http://lbs.amap.com***

```
86		<meta-data
87			android:name="com.amap.api.v2.apikey"
88			android:value=Your key in GaoDe Map />
```
